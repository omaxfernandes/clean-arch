# Clean Architecture 

## How to Start the Environments

To start the environment:

1. Navigate to the clean-arch folder:

```bash
   cd clean-arch
```

2. Run the command to start the containers:

```bash
   docker-compose up -d
```

3. Run the command to stopped containers:

```bash
   docker-compose down
```

### Notes

- Make sure you have Docker installed on your machine before running the commands.
